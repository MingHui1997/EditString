EditString是一个字符串库，可以对字符串进行替换，片段，清除等操作。

==============================使用方法=============================

将EditString文件夹中的源文件添加到你的项目中。
在代码中添加头文件 "EditString.h"。

===================================================================


===============================函数================================
-------------------------------------------------------------------
void strslice(char *dest,char *sour,size_t s,size_t e)

将源字符串中的片段复制到目标字符串。

dest    : 目标字符串
sour    : 源字符串
s       : 片段开始点
e       : 片段结束点

头文件 : "EditString.h"
-------------------------------------------------------------------
-------------------------------------------------------------------
void strclear(char *dest)

清除字符串中的所有字符。

dest    : 目标字符串

头文件 : "EditString.h"
-------------------------------------------------------------------
-------------------------------------------------------------------
size_t charindex(char *dest,char fc)

获取指定的字符在目标字符串中第一次出现的位置。
如果目标字符串中没有指定的字符，返回值为 -1。

dest    : 目标字符串
fc      : 指定的字符

头文件 : "EditString.h"
-------------------------------------------------------------------
-------------------------------------------------------------------
size_t charindex_last(char *dest,char fc)

获取指定的字符在目标字符串中第一次出现的位置。(从后向前查找)
如果目标字符串中没有指定的字符，返回值为 -1。

dest    : 目标字符串
fc      : 指定的字符

头文件 : "EditString.h"
-------------------------------------------------------------------
-------------------------------------------------------------------

dest    : 目标字符串

void strreplace(char *dest,char *repstr,char *sour)

dest    : 目标字符串
repstr  : 需要替换的字符串片段
